Job Management

Create Cracking Job

POST
/api/v1/jobs/
Request Body:
{ "name": "WiFi WPA2 Crack Job", "hash_type": 2500, "attack_mode": 0, "hash_file_id": "uuid-of-uploaded-hash-file", "wordlist": "rockyou.txt" }
Common Hash Types:
0: MD5
100: SHA1
1400: SHA256
2500: WPA/WPA2
22000: WPA-PBKDF2-PMKID+EAPOL
3200: bcrypt
Example Request
curl -X POST -H "Content-Type: application/json" \ -d '{ "name": "WiFi Crack Job", "hash_type": 2500, "attack_mode": 0, "hash_file_id": "123e4567-e89b-12d3-a456-426614174000", "wordlist": "rockyou.txt" }' \ http://localhost:1337/api/v1/jobs/

Complete Examples

WiFi WPA/WPA2 Cracking Workflow

1 Upload handshake file:
curl -X POST -F "file=@wifi_handshake.hccapx" \ http://localhost:1337/api/v1/hashfiles/upload
2 Upload wordlist:
curl -X POST -F "file=@rockyou.txt" \ http://localhost:1337/api/v1/wordlists/upload
3 Create cracking job:
curl -X POST -H "Content-Type: application/json" \ -d '{ "name": "WiFi Network Crack", "hash_type": 2500, "attack_mode": 0, "hash_file_id": "your-hash-file-uuid", "wordlist": "rockyou.txt" }' \ http://localhost:1337/api/v1/jobs/
4 Start the job:
curl -X POST http://localhost:1337/api/v1/jobs/your-job-uuid/start
5 Monitor progress:
curl http://localhost:1337/api/v1/jobs/your-job-uuid
